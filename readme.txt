copy the d3, map and facetview folders to:
<solr_root>/example/solr-webapps/webapps/.

copy the default-ts.json in the banana folder to replace:
<solr_root>/example/solr-webapps/webapps/banana/src/app/dashboards/default.json

copy the index.html in the banana folder to replace:
<solr_root>/example/solr-webapps/webapps/banana/src/index.html

then open the following url in the browser:
http://localhost:<solr_port>/solr/banana/src/index.html
